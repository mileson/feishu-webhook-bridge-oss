# 飞书开放平台配置指南 - 长连接模式

## 重要说明

**如果飞书开放平台显示 "应用未建立长连接"，90% 的原因是订阅方式配置错误！**

### 步骤 6：添加机器人到群聊

1. 在应用管理获取机器人二维码
2. 用飞书扫码添加到群聊
3. 或直接在群聊中搜索机器人名称

## 常见错误

| 错误 | 原因 | 解决方案 |
|-----|------|---------|
| 应用未建立长连接 | 选择了 HTTP 回调模式 | 改为长连接模式 |
| 订阅失败 | 权限未开通 | 开通必需权限 |
| 消息发送失败 | 机器人未加入群聊 | 添加机器人到群聊 |

## 调试建议
