# 参与贡献

感谢你关注这个项目。

## 开发流程

1. 克隆仓库
2. 复制 `.env.example` 为 `.env`（也可以直接运行 `./start.sh codex` 或 `./start.sh claude` 自动生成）
3. 本地调试运行
4. 完成功能或修复后更新相关文档

## 提交前建议

请至少确认以下事项：

- 变更范围聚焦，不顺手修改无关内容
- 代码可以通过基本语法检查
- 如果行为有变化，已同步更新 `README.md` 或 `docs/` 文档
- 没有提交 `.env`、日志、数据库、虚拟环境等本地产物
- 没有提交任何密钥、Token、私有地址、个人账号信息

## Issue 建议内容

提交问题时，建议附带：

- 复现步骤
- 期望结果
- 实际结果
- 关键日志
- 当前 Provider 模式（`codex` 或 `claude`）
- 操作系统与 Python 版本

## Pull Request 建议

- 标题尽量清晰说明修改目的
- 描述中说明为什么要改、改了什么、如何验证
- 如果涉及配置项，请更新 `.env.example`
